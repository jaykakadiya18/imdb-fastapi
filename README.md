## How to run in local env:
- git clone 
```shell
 https://github.com/jaykakadiya18/imdb-fastapi.git
```
- you can use requrenment.txt file OR you can use poetry as well pyproject.toml
```shell
 python3 -m pip install -r requirements.txt
```
run project
```shell
 python main.py
```
